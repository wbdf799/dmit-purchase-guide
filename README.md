DMIT 产品线按网络质量分三档，每个机房都有这三档（部分档位暂时缺货）：

**Premium（Pro 系列）**：顶配。电信联通移动三网全走 CN2 GIA，双向优化，晚高峰延迟最稳。洛杉矶节点实测延迟通常在 155-180ms，丢包率极低。这是 DMIT 的核心产品，也是最贵的档位。

## 怎么登录 DMIT 官网并进入购买流程

1. 打开 DMIT 官网，点击右上角 **Sign Up** 注册账号，或 **Login** 登录已有账号
2. 登录后，顶部导航点击 **Pricing** 或 **Services → Cloud Instances** 进入套餐列表
3. 选择机房（洛杉矶 / 香港 / 东京）和线路系列（Premium / Eyeball / Tier 1）
4. 点击目标套餐的 **Order Now**，进入配置页
5. 选择付款周期（月付 / 季付 / 年付），设置主机名，选择系统镜像
6. 在结算页找到 **Promo Code** 字段，粘贴优惠码，点击校验后提交订单
7. 支付完成后几分钟内会收到开机邮件，首次登录需要用 SSH 密钥，不支持密码登录

注意第 7 步——DMIT 默认只能 SSH 密钥登录，购买后及时下载私钥，密钥只能下载一次，丢了只能重新生成。
